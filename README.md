# Smart Home Electricity Usage Estimator

A simple Python console app to estimate household energy consumption and monthly electricity bill.

## Features
- Add, remove, list appliances
- Compute per-appliance and total monthly energy (kWh)
- Estimate monthly bill using a given rate per kWh
- Show percentage contributions and top consumers
- Save / load appliance data (`appliances.json`)
- Export a plain-text report (`report.txt`)

## How to run
1. Install Python 3.8+.
2. Clone the repository or copy the files.
3. Run:

```bash
python3 electricity_estimator.py
